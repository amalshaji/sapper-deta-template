# Sapper-Deta template

Sapper template with polka replaced with express. 


# Dev mode

```bash
make dev 
# or
npm run dev
```

# Deploy to deta

```bash
make deploy
# or 
npm run build # to build the sapper project into the __sapper__ directory
deta deploy # make sure the project is initialized with deta
```
